GenAI Decision Intelligence

GenAI Decision Intelligence is an AI-assisted multi-criteria decision analysis system designed to help users compare options across different domains using personalized priorities and explainable reasoning.

The system combines:

AI-generated option evaluation
weighted scoring logic
personalized criteria importance
narrative-based reasoning
visual analytics

Users provide:

a decision domain
multiple options
their priorities
importance weights for evaluation criteria

The system then:

evaluates the options using an LLM
calculates weighted decision scores
ranks the options
generates a strategic narrative explaining the recommendation

Supported domains currently include:

Career
Education
Investment
Tech Purchase
Food
Fitness
Fashion

Examples of supported comparisons:

iPhone 16 vs Samsung S26
Startup vs Government Job vs Remote MNC
MBA vs MS CS
Nike vs Adidas vs Converse
Salad vs Pizza vs Protein Bowl

How It Works

User selects a domain.
User enters multiple options.
User defines personal priorities.
User adjusts importance weights for criteria.
AI evaluates each option against the criteria.
The scoring engine computes weighted scores.
The system ranks the options.
Narrative reasoning and visual analytics are generated.
